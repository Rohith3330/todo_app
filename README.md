# todo_app

A new project

## Getting Started

### Prerequisites
- Node.js 16+
- npm or yarn

### Install Dependencies

```bash
npm install
```

### Development

Run both frontend and backend in development mode:

```bash
npm run dev
```

### Build

```bash
npm run build
```

## Project Structure

- `/frontend` - React TypeScript frontend (Vite)
- `/backend` - Node.js/Fastify API server
- `/packages` - Shared libraries (optional)
